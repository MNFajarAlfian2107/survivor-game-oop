# 🎮 Survivor Game OOP

Game survival sederhana melawan monster yang dibuat menggunakan **Python** dan **Pygame** dengan konsep **Object-Oriented Programming (OOP)**.

---

## 📌 Deskripsi
Survivor Game adalah game di mana pemain harus bertahan hidup dari serangan monster. Pemain dapat menyerang, mendapatkan XP, dan naik level seiring permainan berlangsung.

---

## 🚀 Fitur
- Sistem pergerakan karakter
- Sistem serangan (shoot)
- Efek suara (hit, explosion, level up, dll)
- Sistem XP dan leveling
- Musuh (monster) yang menyerang pemain
